Apa itu Javascript? Javascript adalah sebuah bahasa komputer atau kode pemrograman yang digunakan pada website agar website tersebut menjadi lebih interaktif dan dinamis. Javascript populer di internet dan dapat bekerja di sebagian besar penjelajah web populer seperti Google Chrome, Internet Explorer, Mozilla Firefox, Netscape dan Opera. Javascript adalah jenis bahasa pemrograman client side. Penggunaan kode javasript pada sebuah website bersifat opsional, artinya tidak harus selalu ada. Fungsi Javascript pada HTML adalah membuat halaman web lebih interaktif, juga digunakan dalam perangkat lunak,server, dan kontrol perangkat keras tertanam. Beberapa tipe data dalam pemrograman Javasript; 1. String (teks) 2. Integer atau Number (bilangan bulat) 3. Float (bilangan pecahan) 4. Boolean 5. Object
